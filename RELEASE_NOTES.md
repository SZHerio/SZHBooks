# SZHBooks 1.0.0

Обновлено 23 июля 2026 года.
Updated July 23, 2026.

## Русский

SZHBooks 1.0.0 — первый стабильный Windows-релиз локальной настольной читалки.
Обновленная сборка 1.0.0 сохраняет формат профиля и совместимость с уже
установленной версией.

### В составе релиза

- обложечная библиотека с крупными карточками и открытием книги одним кликом
- единое меню под логотипом для добавления книг, OneDrive и настроек
- стабильные контрастные состояния кнопок в белой и черной темах
- русский язык по умолчанию и однократная миграция старых профилей
- жесткая фиксация прокрутки на верхней и нижней границах
- TXT, FB2, EPUB, PDF, HTML, Markdown и DOCX
- коллекции, метаданные, пользовательские обложки и полнотекстовый поиск
- позиции чтения, история переходов, закладки, выделения и заметки
- непрерывный и постраничный режимы, настройки типографики и PDF zoom
- экспорт/восстановление профиля и синхронизация через локальную папку
- автоматические снимки профиля, восстановление и диагностические журналы
- установщик Windows и переносимая ZIP-версия

Переустановка сохраняет профиль в пользовательском каталоге приложения.
Portable-сборка хранит отдельный профиль рядом с исполняемым файлом.

## English

SZHBooks 1.0.0 is the first stable Windows release of the local-first desktop
reader. This refreshed 1.0.0 build preserves the profile format and remains
compatible with existing installations.

### Included

- a cover-focused library with larger books and single-click opening
- one logo menu for adding books, OneDrive, and application settings
- stable, high-contrast button states in both white and black themes
- Russian by default, including a one-time migration of existing profiles
- strictly clamped scrolling at the beginning and end of content
- TXT, FB2, EPUB, PDF, HTML, Markdown, and DOCX
- collections, metadata, custom covers, and full-text search
- reading positions, navigation history, bookmarks, highlights, and notes
- continuous and paged modes, typography controls, and PDF zoom
- profile export/recovery and local-folder synchronization
- automatic profile snapshots, recovery, and diagnostic logs
- installed and portable Windows packages

Reinstalling keeps the profile in the user's application data directory.
Portable builds keep a separate profile beside the executable.

## Known limitations / Известные ограничения

- Android packages are not part of 1.0.0.
- EPUB fixed-layout and DRM-protected books are not supported.
- PDF annotations remain in the SZHBooks profile and are not written into PDFs.
- OneDrive currently depends on its desktop synchronization client; Microsoft
  Graph support is planned for Android.
